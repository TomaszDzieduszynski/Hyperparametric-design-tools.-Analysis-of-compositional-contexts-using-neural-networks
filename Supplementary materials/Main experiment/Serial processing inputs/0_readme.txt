Ten folder zawiera 100 kafelków wejściowych do demonstracji możliwości przetwarzania seryjnego większych obszarów (na przykładzie Warszawy). Kanałem czerwonym oznaczono układ komunikacyjny, zielonym zieleń, niebieskim topografię terenu.
