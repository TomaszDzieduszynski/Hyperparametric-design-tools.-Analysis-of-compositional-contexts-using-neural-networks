Ten folder zawiera 3000 obrazów treningowych po usunięciu dupikatów i po zastosowaniu technik rozszerzenia zbioru treningowego. Po prawej kanałem czerwonym oznaczono układ komunikacyjny, zielonym zieleń, niebieskim topografię terenu. Po lewej kanałem czerwonym oznaczono osie naprowadzające, kanałem białym punkty główne.
Tomasz Dzieduszyński 2024.
