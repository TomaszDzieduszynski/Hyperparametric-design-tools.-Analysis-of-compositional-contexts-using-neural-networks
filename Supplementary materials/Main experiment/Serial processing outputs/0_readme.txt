Ten folder zawiera 100 kafelków wyjściowych (przewidywania wytrenowanej sieci neuronowej) do demonstracji możliwości przetwarzania seryjnego większych obszarów (na przykładzie Warszawy). Kolorem czerwonym sieć oznaczyła osie naprowadzające, kolorem białym przewidywane punkty główne.
