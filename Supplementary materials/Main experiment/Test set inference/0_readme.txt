Ten folder zawiera 160 obrazów testowych wraz z przewidywaniami wytrenowanej sieci. Biały punkt "ground truth" nie oznacza przewidywanego punktu głównego, a jedynie punkt centralny wybranego kadru. Po lewej kanałem czerwonym oznaczono układ komunikacyjny, zielonym zieleń, niebieskim topografię terenu. Po prawej kanałem czerwonym sieć neuronowa oznaczyła przewidywane osie naprowadzające, a biały przewidywane punkty główne.
Tomasz Dzieduszyński 2024.

This folder contains 160 test images along with predictions from the trained network. The white 'ground truth' point does not indicate the predicted key point, but merely the central point of the selected frame. On the left, the communication network is marked in red, greenery in green, and the terrain topography in blue. On the right, the neural network marked the predicted guiding axes in red, and the predicted key points in white.
